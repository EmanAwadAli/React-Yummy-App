# 😋 Yummy App
## 📌 Overview
In this project, I created is a web application that displays a set of recipes with its related details.

## 📥 Installation

###### Clone Project
```
git clone https://github.com/EmanAwadAli/Yummy-App.git
```
###### Install npm packages
```
npm i
```
###### Start your Project
```
npm run dev
```
